Basics

A recurrence relation is a functional relation between the independent variable x, dependent variable f(x), and the differences of various orders of f(x). A recurrence relation is also called a difference equation, and we will use these two terms interchangeably.

Example 1: The equation f(x + 3h) + 3f(x + 2h) + 6f(x + h) + 9f(x) = 0 is a recurrence relation. It can also be written as:

ar+3 + 3ar+2 + 6ar+1 + 9ar = 0
yk+3 + 3yk+2 + 6yk+1 + 9yk = 0

Example 2: The Fibonacci sequence is defined by the recurrence relation ar = ar-2 + ar-1, r ≥ 2, with the initial conditions a0 = 1 and a1 = 1.

Order of the Recurrence Relation:

The order of the recurrence relation or difference equation is defined to be the difference between the highest and lowest subscripts of f(x) or ar = yk.

Example 1: The equation 13ar + 20ar-1 = 0 is a first order recurrence relation.

Example 2: The equation 8f(x) + 4f(x + 1) + 8f(x + 2) = k(x)

Degree of the Difference Equation:

The degree of a difference equation is defined to be the highest power of f(x) or ar = yk.

Example 1: The equation y3k+3 + 2y2k+2 + 2yk+1 = 0 has the degree 3, as the highest power of yk is 3.

Example 2: The equation a4r + 3a3r-1 + 6a2r-2 + 4ar-3 = 0 has the degree 4, as the highest power of ar is 4.

Example 3: The equation yk+3 + 2yk+2 + 4yk+1 + 2yk = k(x) has the degree 1, because the highest power of yk is 1 and its order is 3.

Example 4: The equation f(x+2h) - 4f(x+h) + 2f(x) = 0 has the degree 1 and its order is 2.

Introduction

A recurrence relation is a mathematical expression that defines a sequence in terms of its previous terms. In the context of algorithmic analysis, it is often used to model the time complexity of recursive algorithms.

General form of a Recurrence Relation:
fn = f(an-1, an-2, …., an-k)

where f is a function that defines the relationship between the current term and the previous terms. More precisely, in the case where only the immediately preceding element is involved, a recurrence relation has the form:

un = φ(n, un-1) for n > 0

where φ: ℕ × X → X is a function, where X is a set to which the elements of a sequence must belong. For any u0 ∈ X, this defines a unique sequence with u0 as its first element, called the initial value.

This defines a recurrence relation of first order. A recurrence relation of order k has the form:

un = φ(n, un-1, un-2, ..., un-k) for n ≥ k

where φ: ℕ × Xk → X is a function that involves k consecutive elements of the sequence. In this case, k initial values are needed for defining a sequence.

Recurrence relations can be classified into different types, such as homogeneous and non-homogeneous, each with its unique characteristics and applications. Understanding these concepts is crucial for dealing with recursive structures and dynamic programming.

Homogeneous Recurrence Relations

A homogeneous recurrence relation is a relation in which each term is a function of the preceding terms. It follows the form:

T(n) = a1 * T(n-1) + a2 * T(n-2) + ... + ak * T(n-k)


These relations often appear in problems involving recursive structures, such as computing Fibonacci numbers. The main characteristic of homogeneous relations is that they do not have any additional independent terms.

Non-Homogeneous Recurrence Relations

In contrast, a non-homogeneous recurrence relation includes an additional independent function of n, making the relation of the form:

T(n) = a1 * T(n-1) + a2 * T(n-2) + ... + ak * T(n-k) + f(n)

A non-homogeneous recurrence relation has an associated homogeneous recurrence relation where the function f(n) is zero. The general solution of the non-homogeneous recurrence relation is the sum of two parts:

an = ah + at

where ah is the solution of the associated homogeneous recurrence relation, and at is a particular solution to the non-homogeneous recurrence relation.

To find the particular solution, we consider an appropriate trial solution. For example, if the function f(n) is a polynomial or exponential function, we select a trial solution that matches the form of f(n).

Let f(n) = c * xn and let x2 = Ax + B be the characteristic equation of the associated homogeneous recurrence relation. Let x1 and x2 be the roots of this equation.

  • If x ≠ x1 and x ≠ x2, then at = Axn
  • If x = x1, x ≠ x2, then at = Anxn
  • If x = x1 = x2, then at = An2xn

These cases help determine the form of the particular solution depending on the nature of the roots of the characteristic equation.
